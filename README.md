# personal_burnrate
Calculating my monthly budget and categorizing a list of historical purchases into categories to identify how purchases are spread. 

main.py requires Ollama model named Categorizer to run.  
Modelfile included to create similar model to the one I used. Modelfile has lines marked for updates and should not be used to create a model without editing first.
